# 😷 Face Mask Detection using VGG16

## Overview
This project detects whether a person is wearing a **face mask or not** using **Deep Learning and Computer Vision**.  
The model uses **VGG16 with transfer learning** and **OpenCV** for face detection.

## Features
- Real-time face mask detection
- Face detection using OpenCV
- Binary classification: **Mask / No Mask**
- Implemented using **TensorFlow / Keras**

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Google Colab
